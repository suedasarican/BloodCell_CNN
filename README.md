# BloodCell-AI: Kan Hücresi Sınıflandırma
Bu proje, BloodMNIST veri seti kullanılarak kan hücrelerini sınıflandırmak için geliştirilmiş bir Evrişimli Sinir Ağı (CNN) uygulamasıdır.

## Kurulum ve Çalıştırma
1. Gerekli kütüphaneleri kurun: `pip install numpy opencv-python gradio matplotlib seaborn`
2. Modeli eğitmek için: `python train.py`
3. Arayüzü başlatmak için: `python serve.py`
